
# NY Airport Data Warehouse Project
## Overview
Analysed 10+ years of NY airport data using ETL and SQL. Created a central database table (Star Schema) for reporting and visualized metrics using Tableau dashboards.

## Tools & Technologies
- Oracle Apex (SQL)
- Tableau
- Excel

## Key Tasks
- ETL: Extracted, transformed, and loaded data
- Database design: Star Schema for central table
- Data visualization: Created interactive dashboards
- Documentation: Described workflow and insights
